NAME

EiskaltDC++ Wt - GUI based on Wt (Web Toolkit)  

SYNOPSIS

eiskaltdcpp-wt --http-port=<port> --http-address=<address> --docroot <path>
eiskaltdcpp-wt <Key>  

DESCRIPTION

EiskaltDC++ is a program for UNIX-like systems that uses the Direct Connect and ADC protocol. It is compatible with other DC clients, such as the original DC from Neomodus, DC++ and derivatives. EiskaltDC++ also interoperates with all common DC hub software.  

COMMAND LINE OPTIONS

A summary of options is included below.
-h, --help
display help and exit
-v, --version
output version information and exit
 

EXAMPLES

eiskaltdcpp-wt --http-port=8080 --http-address=0.0.0.0 --docroot /usr/share/eiskaltdcpp/wt/
 

FILES

/usr/share/eiskaltdcpp/wt
Directory with resource files
